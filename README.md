# Saikat Ghosh | Professional Portfolio

A modern, high-performance professional portfolio showcasing my experience as a Java Developer and Full-Stack Engineer. Built with a focus on immersive UI, fluid animations, and premium aesthetics.

## Features

- **Interactive 3D WebGL Background**: A continuous, liquid-like organic background powered by `three.js`, `@react-three/fiber`, and custom GLSL shaders that responds to mouse movement and scroll position.
- **Dynamic Glassmorphism**: Premium frosted-glass `.glass-section` UI panels that blur and blend perfectly with the vibrant 3D elements underneath.
- **Immersive Theming**: Comprehensive Light/Dark mode toggling featuring an interactive "X-ray" clip-path onboarding animation.
- **Responsive Architecture**: Built with React, Vite, Framer Motion, and Tailwind CSS for exceptional cross-device scaling.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
- **Icons**: React Icons

## Getting Started

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the local development server

## Deployment

Optimized and ready for production deployment on platforms like Netlify, Vercel, or GitHub Pages. Run `npm run build` to generate the static files.
